Summary of Steps
Install Dependencies: Install the required libraries via pip.
Define Models: Create the Generator and Discriminator models in models/gan_model.py.
Train the GAN: Run scripts/train.py to train the GAN with your dataset.
Generate Images: Use scripts/generate.py to generate new abstract images.
Post-Process (Optional): Use scripts/postprocess.py to add glitch effects or other customizations to the generated images.
